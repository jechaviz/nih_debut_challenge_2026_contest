# Evidence Table

## Clinical and design evidence

| Source | What it supports | RenalCue takeaway |
| --- | --- | --- |
| CDC CKD data | CKD is common and often undiagnosed in U.S. adults. | The clinical need is significant and population-scale. |
| NIDDK UACR/GFR quick reference | UACR and eGFR are key CKD markers. | The MVP is grounded in core kidney measurements. |
| NIDDK CKD evaluation guidance | eGFR alone is not enough to identify progression risk. | RenalCue combines eGFR, UACR and context. |
| KDIGO 2024 CKD guideline | CKD care should use GFR, albuminuria and risk-based planning. | Risk-band routing is guideline-aligned. |
| CKD CDS trials | CDS succeeds only when it changes workflow and action. | RenalCue focuses on visible handoff receipts, not hidden alerts. |
| DEBUT/VentureWell guidelines | Judges value clinical need, impact, innovation and prototype. | The packet maps evidence to each review axis. |
